# 🧾 Paycheck Calculator

Estimate your take-home pay after federal, state taxes, and deductions.

## 🌐 Live Site

**→ [PaycheckWiz.com](https://paycheckwiz.com)**

## ✨ Features

- **Federal Tax Calculation** — 2025/2026 tax brackets
- **State Tax Estimation** — all 50 US states
- **Deductions** — 401(k), health insurance, HSA, FSA
- **Pay Frequency Options** — weekly, biweekly, semi-monthly, monthly
- **Detailed Breakdown** — gross pay, taxes, deductions, net pay

## 📊 Tax Bracket Reference (2026)

| Taxable Income | Federal Rate |
|---|---|
| $0 – $11,925 | 10% |
| $11,926 – $48,475 | 12% |
| $48,476 – $103,350 | 22% |
| $103,351 – $197,300 | 24% |
| $197,301 – $250,525 | 32% |
| $250,526 – $626,350 | 35% |
| $626,351+ | 37% |
